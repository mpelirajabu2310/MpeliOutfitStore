 STEP 1: IMPORT DATABASE
================================================================================

Option A - Command Line (RECOMMENDED):
  cd C:\xampp\htdocs\MpeliOutFitStore
  mysql -u root -p < database_clean.sql
  
  (Enter your MySQL password when prompted)

Option B - phpMyAdmin:
  1. Go to http://localhost/phpmyadmin
  2. Click "Import" tab
  3. Click "Choose File" and select database_clean.sql
  4. Click "Go"

Option C - MySQL Workbench:
  1. Open MySQL Workbench
  2. File → Run SQL Script
  3. Select database_clean.sql
  4. Execute

 STEP 2: VERIFY DATABASE
================================================================================

After import, verify the database was created:

================================================================================
 TROUBLESHOOTING
================================================================================

Q: "mysql: command not found"
A: Add MySQL to your PATH or use full path:
   C:\xampp\mysql\bin\mysql -u root -p < database_clean.sql

Q: "Access denied for user 'root'"
A: Add -p flag or specify password:
   mysql -u root -pYOUR_PASSWORD < database_clean.sql

Q: "Database already exists"
A: This is fine! The script will drop and recreate it.

Q: "Can't find database_clean.sql"
A: Make sure you're in the right directory:
   cd C:\xampp\htdocs\MpeliOutFitStore

Q: "No tables appear after import"
A: Check MySQL service is running:
   net start MySQL80 (Windows)
   or restart your MySQL service
